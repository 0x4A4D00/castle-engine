ABOUT libpng*.a
---------------

*.a files were compiled from libpng sources downloaded from www.libpng.org
Version used: 1.6.7
libpng is open source project, license can be found at
http://www.libpng.org/pub/png/src/libpng-LICENSE.txt

libpng-ios-arm.a and libpng-ios-sim.a static libraries were compiled using Xcode
and are used in Castle Game Engine in cases when dynamic libraries are not
possible.
